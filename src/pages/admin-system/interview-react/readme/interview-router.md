# 前端路由

## (一) URL 和 URI 的区别
URL -> Locator -> 统一资源定位符 -> 强调地址即定位
URI -> Identifier -> 统一资源标识符 -> 强调标识资源，具有唯一性

## (二) URL的组成
- `http://www.baidu.com:80/stu/index.html?name=xxx&age=25#teacher`
- Protocol：协议
  - `http://`
  - `https://`
- Domain: 域名
  - `www.baidu.com`
- Port：端口 `:80`
  - http协议的默认地址是 `:80`
  - https协议的默认地址是 `:443`
